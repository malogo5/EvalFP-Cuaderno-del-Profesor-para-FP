# Rúbrica — RA1: Instala sistemas operativos, relacionando sus características con el hardware del equipo y el software de aplicación.

**Módulo:** Implantación de Sistemas Operativos  
**Ponderación:** 13%

| Criterio | No Alcanzado (0-4) | En Proceso (5-6) | Alcanzado (7-8) | Sobresaliente (9-10) |
|----------|---------------------|-------------------|------------------|-----------------------|
| CR1      | No identifica las características del sistema operativo ni su relación con el hardware. | Muestra una comprensión básica de las características del sistema operativo, pero no establece conexiones claras con el hardware. | Identifica adecuadamente las características del sistema operativo y las relaciona correctamente con el hardware correspondiente. | Relaciona de manera precisa y detallada las características del sistema operativo con el hardware, demostrando una comprensión profunda. |
| CR2      | No realiza la instalación del sistema operativo o comete errores críticos que impiden su funcionamiento. | Realiza la instalación con errores menores que afectan el rendimiento del sistema, pero logra completar el proceso. | Instala el sistema operativo correctamente, siguiendo los pasos establecidos y asegurando su funcionalidad. | Realiza la instalación de manera eficiente y optimizada, anticipando posibles problemas y aplicando soluciones adecuadas. |
| CR3      | No configura las opciones del sistema operativo ni personaliza la instalación según las necesidades del hardware. | Realiza configuraciones básicas, pero carece de personalización según el hardware y el uso previsto. | Configura y personaliza el sistema operativo de manera adecuada, ajustándolo a las características del hardware y necesidades de uso. | Hace configuraciones avanzadas y personalizadas que optimizan el rendimiento del sistema operativo en función del hardware y las aplicaciones. |
| CR4      | No documenta el proceso de instalación ni las configuraciones realizadas. | Documenta de forma incompleta el proceso de instalación y configuraciones, faltando detalles importantes. | Documenta adecuadamente el proceso de instalación y las configuraciones realizadas, proporcionando información clara y útil. | Ofrece una documentación exhaustiva y detallada del proceso de instalación y configuraciones, incluyendo justificaciones de las decisiones tomadas. |
| CR5      | No evalúa el rendimiento del sistema operativo tras la instalación. | Realiza una evaluación básica del rendimiento, pero sin mediciones concretas ni análisis detallado. | Evalúa el rendimiento del sistema operativo utilizando herramientas adecuadas y analiza los resultados obtenidos. | Realiza una evaluación exhaustiva del rendimiento, utilizando múltiples herramientas y proponiendo mejoras basadas en el análisis detallado. |
| CR6      | No identifica ni soluciona problemas que surgen durante la instalación. | Reconoce algunos problemas durante la instalación, pero no proporciona soluciones efectivas. | Identifica y soluciona problemas comunes que surgen durante la instalación del sistema operativo de manera efectiva. | Diagnostica y resuelve problemas complejos con creatividad y eficacia, mejorando el proceso de instalación en su totalidad. |
| CR7      | No considera la compatibilidad entre el sistema operativo y el software de aplicación. | Muestra una comprensión limitada de la compatibilidad, pero no aplica criterios claros en la instalación. | Considera la compatibilidad entre el sistema operativo y el software de aplicación, asegurando su correcta instalación y funcionamiento. | Evalúa y garantiza la compatibilidad de manera exhaustiva, eligiendo software óptimo que maximiza el rendimiento del sistema operativo. |
| CR8      | No realiza pruebas de funcionalidad del sistema operativo tras la instalación. | Realiza pruebas de funcionalidad básicas, pero con un enfoque limitado y sin un análisis crítico. | Realiza pruebas de funcionalidad del sistema operativo, garantizando que todas las características funcionan correctamente. | Lleva a cabo pruebas exhaustivas de funcionalidad y rendimiento, proporcionando un análisis crítico y sugerencias de mejora. |
| **NOTA GLOBAL** | No muestra comprensión ni aplicación de los conceptos básicos relacionados con la instalación de sistemas operativos. | Muestra un entendimiento básico, pero con múltiples áreas de mejora en la instalación y configuración de sistemas operativos. | Demuestra comprensión y ejecución adecuada de la instalación y configuración de sistemas operativos con un buen nivel de detalle. | Muestra un dominio excepcional en la instalación y configuración de sistemas operativos, con un enfoque analítico y proactivo hacia la optimización y solución de problemas. |
