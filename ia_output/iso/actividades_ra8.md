# Propuestas de Actividades — RA8: Establece la conexión en red de los sistemas informáticos, configurando dispositivos y protocolos.

**Módulo:** Implantación de Sistemas Operativos

### Actividad 1: Configuración de una Red Local

1. **Nombre**: Creación y Configuración de una Red Local
2. **Objetivo**: Aprender a establecer y configurar una red local utilizando dispositivos de red y protocolos adecuados.
3. **Descripción**: En un entorno simulado, los alumnos recibirán un diseño básico de una red local que incluye múltiples dispositivos (PCs, impresoras y un servidor). Deberán configurar los dispositivos de red, establecer las direcciones IP, y asegurarse de que todos los equipos puedan comunicarse entre sí. Al finalizar, presentarán un documento que detalle el proceso de configuración y los problemas encontrados. Se simulará un entorno de trabajo real con roles asignados (administradores de red, técnicos de soporte).
4. **Criterios cubiertos**: CR1, CR2, CR3, CR4, CR6.
5. **Instrumento de evaluación**: Proyecto.
6. **Duración estimada**: 6 horas.
7. **Recursos necesarios**: Equipos informáticos, routers, switches, cables de red, software de simulación de redes (por ejemplo, Cisco Packet Tracer), y documentación de protocolos.

### Actividad 2: Diagnóstico y Solución de Problemas de Conexión

1. **Nombre**: Resolución de Problemas de Conexión en Red
2. **Objetivo**: Desarrollar habilidades para diagnosticar y solucionar problemas de conexión en una red existente.
3. **Descripción**: Se presentará a los alumnos un caso real donde una red presenta problemas de conectividad. Utilizando herramientas de diagnóstico (ping, traceroute, etc.), deberán identificar la raíz del problema, que puede incluir fallos en dispositivos, configuraciones incorrectas o problemas de protocolo. Los alumnos documentarán el proceso de diagnóstico y las soluciones aplicadas, presentando un informe final que incluya los pasos dados y las herramientas utilizadas.
4. **Criterios cubiertos**: CR5, CR6, CR7, CR8.
5. **Instrumento de evaluación**: Práctica.
6. **Duración estimada**: 4 horas.
7. **Recursos necesarios**: Ordenadores, conexión a una red simulada, herramientas de diagnóstico (software y hardware).

### Actividad 3: Implementación de Seguridad en Redes

1. **Nombre**: Configuración de Seguridad en Redes Locales
2. **Objetivo**: Aprender a implementar medidas de seguridad en la red local, configurando dispositivos para proteger la información.
3. **Descripción**: En esta actividad, los alumnos trabajarán en grupos para diseñar y configurar un sistema de seguridad en una red local. Deberán implementar firewalls, protocolos de seguridad (como WPA2 para redes Wi-Fi), y gestionar accesos a través de VLANs. Al finalizar, presentarán un informe que detalle las decisiones de seguridad tomadas, así como las configuraciones realizadas. Se simulará una auditoría de seguridad donde otros grupos evaluarán la solidez de las medidas adoptadas.
4. **Criterios cubiertos**: CR3, CR4, CR6, CR9.
5. **Instrumento de evaluación**: Proyecto.
6. **Duración estimada**: 6 horas.
7. **Recursos necesarios**: Equipos informáticos, routers y switches con capacidades de seguridad, software de configuración de redes, y guías de buenas prácticas en seguridad.
