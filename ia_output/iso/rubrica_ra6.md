# Rúbrica — RA6: Supervisa el rendimiento del sistema, diagnosticando las causas de pérdidas de rendimiento.

**Módulo:** Implantación de Sistemas Operativos  
**Ponderación:** 12%

| Criterio | No Alcanzado (0-4) | En Proceso (5-6) | Alcanzado (7-8) | Sobresaliente (9-10) |
|----------|--------------------|------------------|-----------------|----------------------|
| CR1      | No identifica los principales indicadores de rendimiento del sistema. | Reconoce algunos indicadores de rendimiento, pero no los aplica correctamente. | Identifica y aplica correctamente indicadores de rendimiento adecuados del sistema. | Identifica y aplica de manera exhaustiva y precisa diversos indicadores de rendimiento del sistema. |
| CR2      | No realiza un análisis básico de los registros del sistema para la supervisión del rendimiento. | Realiza un análisis superficial de los registros, pero no saca conclusiones claras. | Analiza de forma adecuada los registros del sistema y extrae conclusiones relevantes. | Realiza un análisis exhaustivo y crítico de los registros, proporcionando recomendaciones basadas en datos. |
| CR3      | No detecta problemas de rendimiento o lo hace de manera incorrecta. | Detecta problemas de rendimiento, pero no ofrece soluciones viables. | Detecta y clasifica problemas de rendimiento, proponiendo soluciones adecuadas. | Detecta problemas complejos de rendimiento y propone soluciones innovadoras y efectivas. |
| CR4      | No utiliza herramientas de diagnóstico de rendimiento. | Utiliza herramientas básicas, pero no de manera efectiva. | Utiliza herramientas de diagnóstico de rendimiento de manera efectiva y con criterio. | Utiliza diversas herramientas de diagnóstico de rendimiento con alta efectividad y presenta análisis comparativos. |
| CR5      | No realiza pruebas de rendimiento y no comprende su importancia. | Realiza pruebas básicas, pero sin un enfoque claro. | Realiza pruebas de rendimiento de manera adecuada, interpretando los resultados. | Realiza pruebas exhaustivas de rendimiento, analizando resultados y proponiendo mejoras. |
| CR6      | No establece umbrales para la evaluación del rendimiento del sistema. | Establece umbrales básicos, pero no fundamentados. | Establece umbrales adecuados y fundamentados para la evaluación del rendimiento. | Establece y justifica umbrales críticos que permiten un monitoreo proactivo del rendimiento. |
| CR7      | No elabora informes sobre el rendimiento del sistema. | Elabora informes poco claros y sin análisis crítico. | Elabora informes claros y concisos sobre el rendimiento, incluyendo análisis. | Elabora informes detallados y perspicaces, con análisis profundo y recomendaciones específicas. |
| CR8      | No comunica las conclusiones sobre el rendimiento de manera efectiva. | Comunica conclusiones de manera confusa y poco estructurada. | Comunica conclusiones de forma clara y estructurada, adecuadas para el público objetivo. | Comunica conclusiones de manera excepcional, utilizando lenguaje técnico y accesible, adaptándose al público. |
| CR9      | No realiza seguimiento de las acciones correctivas implementadas. | Realiza seguimiento limitado y poco riguroso de las acciones correctivas. | Realiza seguimiento adecuado y riguroso, evaluando la efectividad de las acciones correctivas. | Realiza un seguimiento exhaustivo y crítico, ajustando acciones en base a resultados y proponiendo nuevas estrategias. |
| CR10     | No evalúa el impacto de las intervenciones en el rendimiento del sistema. | Evalúa de manera limitada el impacto de las intervenciones. | Evalúa adecuadamente el impacto de las intervenciones sobre el rendimiento del sistema. | Evalúa de manera exhaustiva el impacto, proporcionando análisis detallados y recomendaciones para futuras intervenciones. |
| **NOTA GLOBAL** | El alumno no demuestra comprensión ni aplicación de los conceptos necesarios para supervisar el rendimiento del sistema. | El alumno muestra comprensión básica, pero su aplicación es limitada y poco efectiva. | El alumno demuestra una buena comprensión y aplicación de los conceptos, logrando resultados satisfactorios. | El alumno muestra una comprensión profunda y aplica los conceptos de manera sobresaliente, logrando resultados excepcionales. |
