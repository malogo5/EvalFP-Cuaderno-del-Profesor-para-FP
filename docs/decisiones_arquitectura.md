# Decisiones de Arquitectura (ADR)

## ADR-001

Un archivo representa un curso académico. Aceptada.

## ADR-002

Un profesor trabaja sobre un solo archivo. Aceptada.

## ADR-003

El motor se basa en tablas estructuradas. Aceptada.

## ADR-004

No se usan dependencias de Microsoft Access. Aceptada.

## ADR-005

La compatibilidad con Windows y macOS es obligatoria. Aceptada.

## ADR-006

No hay complementos externos obligatorios. Aceptada.

## ADR-007

La arquitectura está preparada para múltiples módulos. Aceptada.
