# Decisiones de arquitectura

## ADR-001

Fecha

01/07/2026

Decisión

No corregir mediante un parche el cambio del número de evaluaciones.

Motivación

El problema afecta al modelo de datos del Plan de Evaluación y una solución rápida podría provocar pérdida de información y nuevas inconsistencias.

Consecuencia

Se desarrollará un nuevo modelo para el Plan de Evaluación en la versión 3.1.
